Help Page

Add task

To add a task, navigate to the home page from the top or click Add Task in the side bar. You will directed to the task input where you can fill out the neccessary fields. Below is a sample entry using some of the fields. To save the entry just simply click the add button on the home page.

Note: All fields are not mandatory, but a default value will be saved if it is blank.


Pop-up Tasks

After you add the task, it will pop up in the right side bar of the home page. Below is an example of task we just created would look like. From here you have the option of editing the existing information by pressing the edit button. This will display yours existing entries in the form. You can also press the done button which will complete the task and it will disappear from the page. It will be saved to the completed tasks. The last option is to delete the task using the delete button.

If you click on the task name it will direct you to a more detailed page of the specific task information you can see. This is explained more in the next section.

Task Details

Once you've clicked on the task name, you will be directed to the detail page where all the information you entered will be diplayed including the description.

From here you have the option of completing the task using the done button or deleting the task using the delete button. If you are done reviewing the details, the back buttom will redirect you to the home page.

Dashboard Lists

After clicking on the subheading you want, a list of tasks like the one below will popup. This will give all the information of the task except the description, which can be found on the detail page. Click on the task name list redirect you to the detail page.

From the list you can complete a task or just see what you have scheduled for the day.



Calendar Page

To go to the calendar page you just need to click on the calendar tab in the left side bar. There the calendar will show you which tasks you have scheduled on each day, with links to the detail page.
